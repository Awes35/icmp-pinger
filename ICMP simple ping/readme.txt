In order to run/compile the program,
1) In terminal, when in the folder location of the program file, type the command 	"python3" followed by the file name, and then followed by the flag 
	"--target-host" and finally the address of the target.

	ex: "python3 IcmpPingSkel-1.py --target-host www.xavier.edu"

Output:
The program will ping/pong, and display timings/results for the specified host address.

There should be 4 pings, each displaying its "pong" or reply back from the host.
